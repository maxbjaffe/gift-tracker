## Technical Features & Architecture

### **Frontend**
- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS with custom GiftStash color palette
- **UI Components**: Radix UI + shadcn/ui for accessibility
- **State Management**: React hooks, TanStack Query for data fetching
- **PWA**: next-pwa with Workbox for service workers

### **Backend**
- **Database**: Supabase (PostgreSQL) with Row Level Security
- **Authentication**: Supabase Auth with email/password and social auth
- **Real-time**: Supabase realtime subscriptions for live updates
- **SMS**: Twilio for SMS sending and receiving
- **AI**: Anthropic Claude for natural language processing and recommendations
- **Webhooks**: Twilio webhooks for incoming SMS processing

### **Data Models**

#### **Users/Profiles**
- user_id, email, phone_number
- sms_onboarded (boolean)
- budget_settings (JSON)
- created_at, updated_at

#### **Recipients**
- id, user_id, name, birthday
- relationship, interests, preferences
- avatar, notes
- budget_limit

#### **Gifts**
- id, user_id, name, description
- current_price, category, status
- url, image_url
- source (sms/extension/web)
- source_metadata (JSON with SMS text, photos, etc.)
- created_at, updated_at

#### **Gift-Recipient Associations**
- gift_id, recipient_id
- Many-to-many (one gift can be for multiple people)

### **Security & Privacy**
- **Row Level Security**: Users only see their own data
- **Encrypted SMS**: Twilio secure transmission
- **No Data Selling**: User data never shared or sold
- **GDPR Compliant**: Right to deletion, data export
- **Secure Authentication**: Industry-standard practices
