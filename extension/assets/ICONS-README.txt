EXTENSION ICONS SETUP

You need to create 3 icon files before loading the extension:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

OPTION 1: Use the HTML Generator (Easiest)
1. Open create-icons.html in any browser
2. Right-click each canvas image
3. Select "Save image as..."
4. Save with the exact filename shown (icon-16.png, icon-48.png, icon-128.png)
5. Save all icons in this /assets folder

OPTION 2: Create Your Own Icons
- Create 16x16, 48x48, and 128x128 PNG images
- Use the purple/pink gradient (#9333ea to #ec4899)
- Add a gift box or gift icon
- Name them icon-16.png, icon-48.png, icon-128.png
- Place in this /assets folder

OPTION 3: Use Existing Images
If you have any square PNG images, you can temporarily use those:
- Rename them to icon-16.png, icon-48.png, icon-128.png
- Place in this /assets folder
- The extension will work, just with different icons
